# 洛克王国：世界 叶天帝 自动鞠躬骑乘脚本

完全免费，`exe` 脚本请在仓库 `Release` 中下载。

## 功能

- 图形界面启动、暂停/继续、终止脚本
- 按设定流程执行 `Tab -> 动作键 -> ESC -> R -> 左键`
- 支持随机延迟范围
- 支持单次小循环次数和大循环等待
- 支持保存设置和恢复默认设置
- 支持可选的 `4:00` 月卡跳过点击
  - 每次小循环都会检查北京时间；仅当时间落在 `03:58:00` 到 `04:00:00` 之间时，才会启动等待，并在 `04:00:10` 连续自动点击 `3` 次，每次点击后等待 `5` 秒

## 使用说明

- 启动脚本后，保持游戏窗口为前台
- 如果游戏使用管理员权限运行，本程序也请使用管理员权限运行
- 配置会保存在程序目录下的 `auto_bow_mount_config.json`
